How-to guides

Step-by-step guides for using the Gambling Data Finder as a CYP-GHRIP evidence mapping and data discovery resource.

How to search for gambling variables by keyword v
  1. Use the search box on the home page or open the Search page.
  2. Type a keyword such as "PGSI", "lottery", "online", "harm" or "participation".
  3. The search matches variable names, labels, measure names, study names and categories.
  4. Use filters to narrow results by role, study, topic, risk domain, wave or age group.
  5. Open the linked study page to review access routes and source documentation.
How to explore gambling topics and risk factors v
  1. Go to the Explore page.
  2. Expand topic groups such as gambling behaviour, gambling harms or risk/protective factors.
  3. Select a sub-topic to see matching variables across studies.
  4. Use the pre-filtered search link to inspect and export the full set of matching records.
How to find studies relevant to adolescents or young adults v
  1. Go to the Search page.
  2. Set the Role filter to "Gambling measure".
  3. Set the Age group filter to a relevant label such as "Adolescents", "11-17" or "16-24".
  4. Review the study, wave and questionnaire information on each result card.
  5. Use the Visualisations page to see wider age group coverage.
How to compare gambling harm measures across studies v
  1. Go to the Measures page.
  2. Review recurring instruments such as PGSI, DSM-derived measures and SOGS-RA.
  3. Click a measure name to search for all variables using that measure.
  4. Compare variable labels, wording, wave information and compatibility notes before planning harmonisation.
How to assess whether a study is ready to use v
  1. Open the Coverage page.
  2. Check whether variable names, documentation, questionnaires, sources and access information are available.
  3. Open the study page to inspect access conditions and documentation links.
  4. Treat partial documentation as a prompt for manual review or contact with the study team.
How to find original questionnaire wording v
  1. Search for the variable or measure of interest.
  2. Open the linked study page.
  3. Check the Questionnaires and Sources & documentation sections for questionnaire PDFs, codebooks or data dictionaries.
  4. If wording is not available, note the gap for follow-up with the data owner or study team.
How to export records for an evidence map or access request v
  1. Search or browse until you find variables, questions or questionnaires of interest.
  2. Click "Add to basket" on each record you want to keep.
  3. Go to the Basket page.
  4. Download your selection as CSV or Excel.
  5. Use the export to support a data access application, evidence map, analysis plan or stakeholder discussion.
